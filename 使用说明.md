# 航班管理系统使用说明

## 1. 系统概述
这是一个基于 C 语言实现的航班管理系统，核心特性如下：

- 数据结构采用内核双向链表（`kernel_list.h`）
- 业务对象采用结构体封装（`User`、`Flight`、`Booking`）
- 功能按模块拆分（用户、航班、预订、管理员、工具）

系统覆盖用户端与管理员端常见功能，并支持多舱位定价、用户折扣、退票规则、联合航班管理。

## 2. 快速开始

### 2.1 编译

Windows + MinGW：

```powershell
gcc -std=gnu11 -Wall -Wextra -Iinclude src/*.c -o flight_system.exe
```

Linux/macOS：

```bash
gcc -std=gnu11 -Wall -Wextra -Iinclude src/*.c -o flight_system
```

### 2.2 运行

Windows：

```powershell
.\flight_system.exe
```

Linux/macOS：

```bash
./flight_system
```

## 3. 默认账号
程序启动后会自动注入测试账号：

- 超级管理员：`admin_all / 123456`（可管理所有航司）
- 航司管理员：`admin_cn / 123456`（仅管理 `ChinaAir`）
- 普通用户：`alice / 123456`（用户组为 `student`）

## 4. 菜单总览

主菜单：

```text
1. 用户注册
2. 用户登录
3. 管理员登录
4. 显示全部航班
0. 退出系统
```

## 5. 用户端使用流程

### 5.1 注册与登录
- 先通过 `1. 用户注册` 创建账号
- 再通过 `2. 用户登录` 进入用户菜单

### 5.2 航班查询
- 精确查询：按航班号、出发地+目的地、出发时间查询
- 模糊查询：
  - 价格段：输入价格后按 `±200` 查询经济舱票价
  - 起飞时间段：输入时间后按 `±30` 分钟查询

### 5.3 预订航班
操作步骤：

1. 输入航班号
2. 输入航司（联合航班必须指定）
3. 选择舱位：`1经济舱 2商务舱 3头等舱`
4. 系统计算并显示实付金额

计费公式：

`实付金额 = 舱位基础价 * 用户折扣 + 燃油费 + 机建费`

### 5.4 取消预订
- 可在“我的预订”中查看后取消
- 退票时：
  - 燃油费、机建费不退
  - 票面可退部分按手续费率扣减（当前 10%）

## 6. 管理员使用流程

管理员登录后可执行：

- 航班录入：新增航班信息（航班号、航线、时间、座位、价格）
- 航班删除：按航班号（可加航司）删除，删除时同步清理预订
- 航班更新：可更新航班号、航线、时间、座位、价格
- 航班显示：查看可管理范围内航班
- 用户管理：增删改查用户

权限规则：

- 超级管理员：可管理所有航司航班
- 航司管理员：仅可管理本航司航班

## 7. 关键业务规则

### 7.1 舱位与价格
- 舱位类型：`经济舱 / 商务舱 / 头等舱`
- 每个航班分别维护三种舱位票价

### 7.2 用户折扣
- `normal`：1.00
- `student`：0.90
- `soldier`：0.85
- `elder`：0.80

### 7.3 固定费用与退票
- 燃油费：`50.0`
- 机建费：`60.0`
- 退票手续费率：`10%`

### 7.4 联合航班
- 允许同一航班号在不同航司下存在
- 使用“航班号 + 航司”唯一定位具体航班记录

## 8. 项目结构与模块职责

```text
include/
  flight_system.h      # 公共结构体、常量、函数声明
  kernel_list.h        # 内核链表实现
src/
  main.c               # 程序入口
  fs_data.c            # 全局链表初始化与释放
  fs_utils.c           # 通用输入输出与工具函数
  fs_user.c            # 用户注册、登录、用户菜单
  fs_flight.c          # 航班查询、录入、更新、删除、展示
  fs_booking.c         # 订票、退票、预订链表处理
  fs_admin.c           # 管理员菜单与用户管理
```

## 9. 常见问题

### 9.1 终端中文显示异常
系统已做 Windows 控制台 Unicode 兼容。如果个别终端仍异常，请关闭后重新打开终端再运行。

### 9.2 同航班号为什么出现多条
这是联合航班场景，不同航司可共享航班号，但价格和座位独立管理。
